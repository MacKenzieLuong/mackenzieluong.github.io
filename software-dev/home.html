Harmony

Nov. 24
Unity | C# | Arduino

Submitted to NatHacks 2024, Harmony is an interactive biofeedback visualization that transforms stress data into calming ocean waves using Unity 6, Arduino, and BioAmp EXG pills. I optimized data thresholds for accuracy and responsiveness, producing a system that bridges technology and wellness by turning physiological input into real-time visual feedback. This was intented as a curious dive into biofeedback hardware and to show that game engines can be used for more than just games.
